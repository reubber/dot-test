# Dot - frontend teste


[DEMO](https://reubber.github.io/dot-test/src/)
### Installation

gulp requires [Node.js](https://nodejs.org/) v4+ to run.

Install the dependencies and start the server.

```sh
$ cd dot-test
$ npm install or yarn
```

#### gulp cmds

```sh
$ gulp
$ gulp imgTask
$ gulp style
```
